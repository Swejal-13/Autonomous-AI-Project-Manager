# Autonomous AI Project Manager

An AI-powered platform that automates project management tasks such as planning, team allocation, and task assignment using intelligent agents.

## Features
- AI-based task planning
- Smart team selection
- Skill matching system
- Automated decision making

## Tech Stack
- Python
- FastAPI
- React
- MongoDB
- Machine Learning

## Project Structure
backend/ - API and AI agents  
frontend/ - React UI  
docs/ - Architecture and documentation  

## Installation

1. Clone repository
2. Install dependencies
3. Run backend
4. Run frontend
